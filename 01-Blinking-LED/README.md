# Project 01 — Blinking LED

## What It Does
Controls an external LED to blink on and off every second 
using an Arduino Uno and a basic resistor circuit.

## What I Learned
- How to wire an LED with a current-limiting resistor
- Why resistors are necessary (without one the LED burns out)
- The difference between the built-in LED and an external circuit
- Basic structure of an Arduino sketch: setup() and loop()
- How digitalWrite() controls voltage on a pin
